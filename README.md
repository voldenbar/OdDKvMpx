# 项目简介

## 核心功能
- 基于 YOLOv8 的城市交通目标检测系统，包含 15 类目标识别：人、停车标志、柱子、汽车、自行车、卡车、狗、公交车、摩托车、反光锥、消防栓、警示柱、垃圾桶、三轮车、球形路障
- 支持 7000 张标注数据集
- 提供 15 类城市交通目标的检测能力
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时显示功能：目标位置、目标总数、置信度等信息
- 支持批量检测与结果查看
- 支持检测结果保存
- 支持自定义置信度和 IOU 阈值
- 支持选择显示标签和原图
- 提供多种检测模型选择

## 适用场景
- 城市交通监控与分析
- 智能交通系统
- 车辆流量统计
- 交通目标识别研究
- 安防监控系统
- 智能驾驶辅助系统开发
- 交通设施检测与维护

## 技术特点
- 基于 YOLOv8 目标检测算法
- 采用 Python 3.9 开发环境
- 集成 OpenCV-Python、PyQt5、Ultralytics、Torch 1.9 等技术栈
- 提供图形化用户界面，操作简便
- 包含完整的训练好的模型及训练结果
- 提供详细的项目运行教程
